# Sampling-boundary distortion in Sharpe ratio evidence from financial panels

This repository contains the public manuscript, technical appendix, aggregate public artifacts, and reproducibility code for the paper.

Public repository: https://github.com/DsChauhan08/dependence-aware-evaluation-of-panel-trading-strategy-evidence

## Main Files

- `journal/MF_sharpe_uvif_main_blinded.pdf`: blinded peer-review manuscript PDF for journals that require separate title pages.
- `journal/MF_sharpe_uvif_title_page.md`: author title page.
- `journal/MF_sharpe_uvif_appendix_blinded.pdf`: blinded technical appendix for supplementary upload.
- `journal/MF_sharpe_uvif_replication_blinded.zip`: anonymized replication materials for double-blind review.
- `arxiv/`: editable LaTeX source and generated manuscript artifacts.
- `repro/`: public-data loaders, simulations, tests, aggregate artifacts, and reproduction instructions.

## Release Archives

`arxiv.tar.gz`, `journal.tar.gz`, `ssrn.tar.gz`, and `repro.tar.gz` mirror the corresponding release folders. Each release subfolder includes a `MANIFEST.sha256` file for integrity checks.

## Redaction Boundary

The manuscript tables, figures, and conclusions use public Kenneth French and AQR data plus simulations only. No nonpublic or workspace-specific production artifacts enter the reported evidence.
